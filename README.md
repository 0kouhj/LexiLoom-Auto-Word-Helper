# LexiRoom 自动背单词系统
- 本软件旨在自动化操作安卓系统背单词操作,解放大学生的双手于背单词系统。
## 环境要求
- Ollama(推荐使用qwen2.5:7b模型,不要用思维链模型)
- 安卓手机的USB调试授权
- Windows10/11 x64
## 安装
- 解压运行exe即可
## 使用教程
### 前置
- 打开安卓手机/平板的开发者选项,打开USB调试功能
- 连接手机与电脑,注意使用全功能数据线
- 在电脑上安装Ollma,预先下载AI模型
### Pages
- 在Page1进行环境检测,手机型号,分辨率,Ollama模型都有显示即可进入下一页
- 在Page2进行软件配置
  - 手机进入背单词界面
  - 电脑点击创建配置,此时会弹出手机屏幕截图,进入坐标标定过程
  - 坐标标定使用`左上-右下`两点标定,请按照`题目-A-B-C-D` **顺序点击,并且框大一点,防止长单词超出范围**
  - 标定结束会弹窗输入配置名称
  - 选择希望使用的AI
  - 点击保存配置,保存后点击加载配置
  ## 只有显示在预览配置中的配置才是正在使用的配置!!!如果没有记得点击保存!!!
  - (可选)点击预览配置来观察配置是否能配合手机
- 在Page3进行循环答题操作
## 软件原理
- 通过 ADB 获取终端图像，利用 RapidOCR 识别题目，结合本地 LLM（Ollama/Qwen）进行智能答题的自动化工具。
- 核心特性：
  - 低侵入性：基于图像识别，无需 Hook 进程。
  - 本地化推理：支持本地 LLM，保护隐私，无 API 开销。
  - 极致优化：打包后体积仅 ~60MB，运行高效。
## 仓库目录
- `app`:旧版无gui版本,使用CUDA+pytorch进行OCR,软件体积大速度快(不再维护)
- `bin`:adb环境
- `src`:目前维护的软件文件夹
  - `core`:包含配置管理,adb管理,llm管理,ocr管理,循环线程操作
  - `gui`:所有gui相关代码
- `models`:历史遗留命名,放置配置初始化,错误码代码
